## üöÄ Installation

The plugin can be installed from within Obsidian.

### Obsidian Community Plugin Browser

- Go to `Settings` -> `Community plugins`
- Make sure `Restricted mode` is **off**
- Click `Browse`
- Search for `APIRequest`
- Click `Install` and then `Enable`
